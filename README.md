# Meters.inGaia.Challenge.API

API com objetivo de calcular valores entre 10 e 10000 em metros quadrados, 
a fim de calcular o valor do imóvel baseado no tamanho em m².

## Versionamento
1.0

## Estrutura

- Desenvolvida em .NET 3.1
- Hospedada na Azure e com alta disponibilidade
- Banco SQL hospedado no Azure com alta disponibilidade
- CI/CD feito também no Azure com deploy automático
- Arquivos de versão alocados e disponíveis no [github](https://github.com/ravergod/Meters.inGaia.Challenge.API)
